# 🎤 Whisper LoRA Speech-to-Text

## About
Fine-tuned OpenAI Whisper model using LoRA (Low-Rank Adaptation) 
for efficient Speech-to-Text transcription.

## Results
- WER: 15.79%
- Base Model: openai/whisper-small
- Fine-tuning: LoRA (only 1.44% parameters trained)
- Dataset: Google FLEURS (English)

## Technologies
| Tool | Purpose |
